**UPI QR Code Generator (Python)**
A simple Python mini project that generates UPI payment QR codes for popular UPI apps like **PhonePe**, **Paytm**, and **Google Pay** using a single UPI ID.
This project helps users quickly create QR codes for digital payments without manually creating links.

**Features**
 Generates UPI payment QR codes
 Supports multiple UPI apps:
   PhonePe
   Paytm
   Google Pay
 Saves QR codes as image files (`.png`)
 
**Technologies Used**
  Python
  `qrcode` library
  PIL (Python Imaging Library)
